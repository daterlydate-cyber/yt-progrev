## О проекте

**YT-Progrev** — это инструмент автоматизации работы с YouTube, построенный на браузерной автоматизации (Selenium + undetected-chromedriver) **без использования YouTube API**. Приложение позволяет:

- 🔥 **Прогревать аккаунты** по заданной тематике: бот смотрит видео, лайкает, подписывается, оставляет комментарии
- 📤 **Автоматически загружать видео** через YouTube Studio прямо в браузере
- 🕵️ **Использовать изолированные профили** как в антидетект-браузере (уникальный fingerprint, User-Agent, прокси для каждого аккаунта)
- 📅 **Планировать задачи** прогрева и постинга по расписанию
- 🖥️ **Управлять всем через удобный GUI** на PyQt5 с тёмной темой

### Технологии

- **Python 3.10+**
- **undetected-chromedriver** — обход детектирования автоматизации
- **Selenium** — браузерная автоматизация
- **PyQt5** — графический интерфейс
- **schedule** — планировщик задач

## Первый запуск

```bash
python main.py
```

При первом запуске приложение автоматически создаст необходимые папки:
- `data/profiles/` — профили браузеров
- `data/cookies/` — файлы cookies
- `data/videos/` — видео для загрузки
- `data/logs/` — логи работы

### Создание первого профиля

1. Откройте вкладку **«Профили»**
2. Нажмите кнопку **«Создать профиль»**
3. Заполните поля:
   - **Имя профиля** — уникальное имя (например, `account_01`)
   - **Прокси** — опционально (формат: `http://user:pass@host:port`)
   - **User-Agent** — оставьте «Авто» для случайной генерации
   - **Язык** — язык интерфейса браузера (например, `ru-RU`)
   - **Разрешение экрана** — выберите из списка
4. Нажмите **«Создать»**

## Прогрев аккаунтов

Прогрев — это процесс «очеловечивания» аккаунта: бот имитирует поведение реального пользователя на YouTube.

### Как работает прогрев

1. **Поиск по ключевым словам** — бот ищет видео по заданным тематикам
2. **Просмотр видео** — смотрит видео случайное время в заданном диапазоне
3. **Лайки** — с заданной вероятностью ставит лайки
4. **Подписки** — подписывается на каналы с заданной вероятностью
5. **Комментарии** — оставляет комментарии из ваших шаблонов
6. **Навигация** — прокручивает ленту, переходит по каналам

### Настройка прогрева

1. Перейдите на вкладку **«Прогрев»**
2. Выберите профиль из списка
3. Введите **ключевые слова** через запятую (например: `кулинария, рецепты, готовка`)
4. Настройте параметры:
   - **Действий за сессию** — сколько видео просмотреть за одну сессию
   - **Длительность просмотра** — мин/макс секунд просмотра одного видео
   - **Вероятность лайка** — от 0.0 до 1.0 (0.3 = 30% видео будут лайкнуты)
   - **Вероятность подписки** — от 0.0 до 1.0
   - **Вероятность комментария** — от 0.0 до 1.0
5. Добавьте **шаблоны комментариев** (по одному на строку)
6. Нажмите **«Старт прогрева»**

### Рекомендации по прогреву

- Начинайте с небольших сессий (5-10 действий)
- Используйте реалистичные шаблоны комментариев
- Давайте аккаунту «отдыхать» между сессиями
- Прогревайте 1-2 недели перед началом постинга

## Автопостинг

Загрузка видео происходит через YouTube Studio прямо в браузере — без API.

### Подготовка видео

1. Поместите видеофайлы в папку `data/videos/`
2. Поддерживаемые форматы: MP4, MOV, AVI, MKV

### Загрузка видео

1. Перейдите на вкладку **«Автопостинг»**
2. Выберите профиль (аккаунт, через который будет загрузка)
3. Нажмите **«Выбрать видео»** и укажите файл
4. Заполните метаданные:
   - **Заголовок** — название видео
   - **Описание** — описание видео
   - **Теги** — через запятую
   - **Превью** — обложка (опционально, JPG/PNG)
   - **Доступ** — публичное / по ссылке / приватное
5. Нажмите **«Загрузить»**

### Шаблоны заголовков и описаний

В `config.json` можно задать шаблоны с переменными:
- `{date}` — текущая дата
- `{time}` — текущее время
- `{keyword}` — ключевое слово тематики

## Настройка прокси

### Форматы прокси

```
http://host:port
http://user:password@host:port
socks5://host:port
socks5://user:password@host:port
```

### Добавление прокси

**Способ 1: Через профиль**
При создании профиля укажите прокси в поле «Прокси».

**Способ 2: Через файл**
1. Создайте файл `data/proxies.txt`
2. Добавьте прокси по одному на строку
3. В `config.json` укажите `"proxy_list_file": "data/proxies.txt"`
4. Выберите режим ротации: `round-robin` или `random`

### Проверка прокси

Приложение автоматически проверяет доступность прокси перед использованием.

## Антидетект-профили

Каждый профиль имеет изолированное окружение:

### Что изолировано в каждом профиле

| Компонент | Описание |
|-----------|----------|
| Chrome user-data-dir | Отдельная папка с данными браузера |
| Cookies | Собственные cookies для каждого аккаунта |
| User-Agent | Уникальный UA (можно задать вручную) |
| Fingerprint | Разрешение, платформа, WebGL и др. |
| Прокси | Свой прокси для каждого профиля |
| Язык | Язык интерфейса браузера |
| Часовой пояс | Временная зона (будущий функционал) |

### Экспорт и импорт профилей

Вы можете перенести профиль на другой компьютер:

1. Выберите профиль в таблице
2. Нажмите **«Экспорт»** — создастся ZIP-файл
3. На другом компьютере: **«Импорт»** → выберите ZIP

ZIP-архив содержит: `profile.json`, cookies, данные Chrome.

## Планировщик задач

Автоматизируйте прогрев и постинг по расписанию.

### Добавление задачи прогрева

1. Перейдите на вкладку **«Планировщик»**
2. Нажмите **«Добавить задачу»**
3. Выберите тип: **«Прогрев»**
4. Укажите профиль и интервал (например, каждые 6 часов)
5. Нажмите **«Добавить»**

### Добавление задачи постинга

1. Нажмите **«Добавить задачу»**
2. Выберите тип: **«Постинг»**
3. Укажите профиль, видеофайл и время публикации (HH:MM)
4. Нажмите **«Добавить»**

### Включение планировщика

Нажмите **«Вкл/Выкл планировщик»** для запуска/остановки фоновых задач.

## Советы по безопасности

### Антибан рекомендации

1. **Используйте реальные прокси** — резидентные прокси лучше, чем датацентровые
2. **Не форсируйте** — начинайте с малого (5-10 действий в день)
3. **Соблюдайте паузы** — между сессиями должны быть перерывы (часы, не минуты)
4. **Разнообразьте действия** — не делайте одно и то же снова и снова
5. **Реалистичные комментарии** — пишите осмысленные комментарии, не спам
6. **Прогрев перед постингом** — не загружайте видео на свежий аккаунт
7. **Один IP — один аккаунт** — не используйте один прокси для нескольких аккаунтов
8. **Не меняйте fingerprint** внутри одной сессии

### Настройки для большей безопасности

В `config.json`:
```json
{
  "warmup_settings": {
    "actions_per_session": 5,
    "watch_duration_range": [30, 120],
    "like_probability": 0.2,
    "subscribe_probability": 0.05,
    "comment_probability": 0.03
  }
}
```
